<h1>
    Payment Gateway API
</h1>

<div>
    <b>
        A fast and secure payment processing gateway that enables merchants to accept various payment methods through a <mark>unified API</mark>.
    </b>
</div>

<div>
    <b>
        <br>
        Key features:
    </b>
</div>

<div>
    <ul>
        <li>
            Multi-payment support (cards, QR, SBP, SIM).
        </li>
        <li>
            Secure authentication with <mark>merchant tokens</mark>.
        </li>
        <li>
            Real-time transaction management.
        </li>
        <li>
            Transaction cancellation functionality.
        </li> 
    </ul>
</div>

<div>
    <b>
        Core functionality:
    </b>
</div>

<div>
    <ul>
        <li>
            Processes payment requests from merchants.
        </li>
        <li>
            Routes transactions to appropriate payment providers.
        </li>
        <li>
            Handles authentication and authorization.
        </li>
        <li>
            Provides transaction status tracking.
        </li>
    </ul>
</div>

<div>
    <b>
        Technical stack
    </b>
</div>

<div>
    <ul>
        <li>
            <mark>FastAPI framework</mark>
        </li>
        <li>
            Pydantic for data validation
        </li>
        <li>
            HTTPX for external API communication
        </li>
        <li>
            JWT Bearer authentication
        </li>
    </ul>
</div>
